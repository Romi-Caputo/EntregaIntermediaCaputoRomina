#NOMBRE
Hospitalapp 2022

Plataforma que permite el ingreso de Pacientes, Doctores y Residencias al Hospital de la Dra. Caputo Romina

#INSTALACION
Instalar Python 3.10
Instalar Visual Code
Instalar Db Browser
Instalar SQLite3
Descargar Repositorio "EntregaIntermediaCaputo"
Abrir con Visual Code:
- Correr nuestro programa: python .\manage.py runserver
- Superuser: Usuario: rominacaputo password: rojedadi32

#USO QUE HACE MI PAGINA
http://127.0.0.1:8000/admin
Ingresando con superuser podemos administar el panel Django, modificando nuestros models: "Pacientes", "Doctores", "Residencias"
Además, crear otros usuarios y delimitar su acción en nuestro panel de administracion.
http://127.0.0.1:8000/hospitalapp
A partir de este link, puede ingresar cualquier usuario, se encontrara con una pagina de inicio de bienvenida al sitio y un menu de los distintos modelos.
Podrá testear funciones de listar pacientes, doctores y residencias. 
Tambien podra realizar la busqueda de pacientes mediante la opcion de buscar pacientes
